# Oblivion Desktop v1.0.5
## What is this?
oblivion is a desktop enviornment for Low end machines!
## Installation
```sh
git clone https://github.com/mioojo/obliviom-de
```
* ./setup will setup the oblivion DE!
```sh
cd oblivion-de/ && chmod +x ./setup && ./setup
```
